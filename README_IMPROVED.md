```bash
cd Qubinode Navigator
npm install

```

Open http://localhost:3000 in your browser.

## Configuration

Create a `.env` file in the root directory:

```env

PORT=3000
NODE_ENV=development

# Optional settings
DATABASE_URL=your_database_url
API_KEY=your_api_key
```
